New Component

Text Input

Page Component

Text indicating what the nurse should input.

Type of input to accept, e.g. "numeric", "alphanumeric", "default", "any".
Default means alphabetical character input only, no numbers.
Any means no restrictions.

Unique id for the input value for this page component.
This will also be the name of the corresponding column in the patient database.

If provided, displayed next to the TextInput, e.g. "cm".

Default value to the text input should start off with.
The type of this depends on type property.

Button

Page Component

Text displayed inside the button, should indicate what happens on press.
E.g. "Next Page", "Skip", "Previous", etc.

Displayed above the button, should describe in more detail when/why to press.

Page ID of the page this button will open on press.

Media Item

Page Component

The file name of the image or video to display.
This will have to be uploaded to the workflow via the admin page.

Optional text to display underneath the image or video.

Multiple Choice

Page Component

Text indicating what the nurse should input.

If true, multiple choice can be selected, so the choices will be checkboxes.
Otherwise, only one option may be selected, so the choices will be radio buttons.

Unique id for the input value for this page component.
This will also be the name of the corresponding column in the patient database.

Choice

Subcomponent

Comparison

Logic Component

The type of comparison to be done. The left side of the operation
will be the target value, and the right will be the threshold. For example:

If type is ">" threshold is 40 and targetValueID is "value_01", then
the comparison would be: is value_01's value > 40?

Threshold the target value will be compared to.

Unique id for some input value in another component on this page.
This value will be checked against the threshold.

On pressing the "next" button, if the target value satisfies the threshold,
the page that should be navigated to.

On pressing the "next" button, if the target value doesn't satisfy the threshold,
the page that should be navigated to.
